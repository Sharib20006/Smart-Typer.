# Smart Typer Pro 🚀

Smart Typer Pro is a comprehensive, feature-rich note-taking application and powerful text editor designed for maximum writing efficiency. It seamlessly unifies advanced AI writing assistance, intelligent hybrid voice typing, quick translation tools, and real-time knowledge lookups into a modern, fluid interface.

---

## ✨ Features

### 1. Smart Notes & Folder Management
* **Flexible Creation:** Instantly create organized directories, standard text notes, or dedicated blog posts right from the top-right toolbar.
* **Quick Actions:** Long-click any folder or note in the file browser list to access deep file management tools: *Edit, Run, Read, Share, Move, Rename, and Delete*.
* **Instant Filter:** Tap the Search icon in the toolbar to quickly filter through your files and folders on the fly.

### 2. Hybrid Voice Typing
* **Offline-First Recognition:** Tap the Mic button in the editor to start dictating. Smart Typer Pro prioritizes secure, data-saving offline speech recognition, automatically falling back to online speech recognition if offline services are unavailable.
* **Custom Voice Dictionary:** Long-press the Mic button to quickly select your favorite languages or configure custom voice dictionary replacements to fix recurring dictation quirks.

### 3. AI-Powered Refinement (Gemini 3)
Tap **"Correct with AI"** (located conveniently next to your line and word counters) to instantly process your text using one of four advanced AI writing modes:
* **Default Mode:** Performs deep grammatical, punctuation, and spelling corrections.
* **Formalise and Translate:** Polishes the professional tone of your writing while automatically translating the output to English.
* **Intelligent Contextual Writer:** Intelligently adapts your writing tone and actively fact-checks context, making it perfect for articles or blog posts.
* **Hinglish Script Normalizer:** Perfect for mixed English/Hindi inputs. It smartly maps Hindi phonetics to Devanagari script and English words to Roman script.

### 4. Direct Translation & Reference Lookup
* **Integrated Wikipedia Search:** Open Wikipedia directly from the navigation drawer or the editor's options menu to look up quick facts and paste reference materials directly into your editor.
* **Multi-Engine Translation:** Translate text smoothly across multiple languages using high-accuracy translation pipelines powered by Google and Microsoft.

---

## ⚙️ Settings & Customization

Tailor the app to fit your exact workflow via the centralized Settings panel:
* **Visual Themes:** Smoothly toggle between Light and Dark mode interfaces.
* **Voice Profiles:** Set your default preferred language for voice typing and manage your custom voice dictionary rules.
* **AI Preferences:** Choose a default preferred mode for AI corrections, or set it to "Ask every time" for maximum flexibility.
* **Recent Files Cache:** Access a quick list of your recent documents, and specify exactly how many recent files to keep tracked in your history.

---

## 🛠️ Tech Stack & Architecture

Smart Typer Pro is built using modern Android development practices:
* **Language:** [Kotlin](https://kotlinlang.org/)
* **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose) — for a fully declarative, native, and reactive UI experience.
* **Core AI Engine:** Gemini 3 API
* **Translation Services:** Google Translation API & Microsoft Translator API
* **Data & Reference:** Wikipedia API
